How it works

1) Upload your flat file

Start with a CSV export of pipeline + account metrics (no PHI). The MVP ingests common fields and normalizes them.

2) Generate indicators

We compute leading indicators and a simple risk score using transparent rules while we iterate toward ML models.

3) Act on insights

See at‑risk deals, top drivers, and recommended next steps that align sales + clinical action.

MVP features

  • CSV upload → risk score + contributing factors
  • Deal/Account heatmap of leading indicators
  • Simple, readable logic today; ML roadmap tomorrow
  • Zero PHI by design; works with synthetic or anonymized data
  • Use-case focus: orthopedics & bracing (expandable)
  • Founder-led onboarding + no-cost pilot conversations
  • Roadmap: what‑if scenarios, recommendations, API
  • Deployment targets: web app (Vercel) + secure invite links
